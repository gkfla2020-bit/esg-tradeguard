# 7. 향후 계획 및 로드맵

## 단기 (1~2주)

| 항목 | 상태 | 비고 |
|------|------|------|
| 실제 CNN 모델 학습 | 미완 | 현재 RGB 근사, PyTorch U-Net 학습 필요 |
| DDS 보고서 PDF 형식 | 미완 | 현재 .txt, reportlab으로 PDF 가능 |
| Vercel 배포 | 미완 | 빌드 정상, 배포 설정만 하면 됨 |
| 업종별 서류 분기 | 미완 | HS코드 기반 필수서류 동적 변경 |

## 중기 (1~2개월)

| 항목 | 설명 |
|------|------|
| 실제 위성사진 교체 | 산림 훼손이 눈에 보이는 고해상도 영상 |
| 다국어 지원 | 영문 보고서 생성 (EU 제출용) |

| 사용자 인증 | 로그인/팀 관리/케이스 이력 |

## 장기 (3~6개월)

| 항목 | 설명 |
|------|------|
| 실시간 위성 모니터링 | 주기적 Sentinel-2 스캔 + 알림 |

| 규제 업데이트 자동 반영 | EU 관보 모니터링 → 규칙 자동 갱신 |
| B2B SaaS 론칭 | 유료 구독 모델 |

## 논문/발표 계획
- **핵심 주장**: CNN 기반 산림전용 탐지의 EUDR 적용 가능성
- **방법론**: U-Net Segmentation + Grad-CAM + NDVI 시계열
- **차별점**: 기존 연구(단순 NDVI 임계값)와 달리 CNN으로 원인(팜유 플랜테이션)까지 식별
- **데이터**: Sentinel-2 L2A, Central Kalimantan, 2019~2024

3. 시연 시 강조점 — 기술(CNN) vs 비즈니스(절감액) vs 규제 도메인?
