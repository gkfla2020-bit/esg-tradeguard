# 6. 시연 시나리오

## 시연 플로우 (약 5분)

### 0. 시작 화면 (10초)
- "EU 환경 규제 대응을 위한 통합 컴플라이언스 플랫폼"
- 5단계 파이프라인 stepper 표시
- "시작하기" 클릭

### 1. 서류 접수 (40초)
- "샘플 케이스 불러오기" 클릭 → 6개 서류 자동 업로드
- "서류 검증 시작" 클릭 → 검증 완료
- 자동 인식 결과: PT. Sawit Kalimantan Utama, HS 1511.10, CPO 2,400MT
- 적용 규제: EUDR + CBAM + CSDDD 자동 매핑

### 2. OCR 추출 (40초)
- "OCR 추출 실행" 클릭
- 12개 필드 순차 출현 (Exporter, HS Code, GPS 좌표 등)
- 교차 검증 결과 (ALL MATCH)
- "이 정보가 맞습니까?" 확인 → 다음

### 3. CBAM 비용 분석 (60초)
- 자동 흐름: 품질 스코어링(85점) → 블록체인 인증 → 시뮬레이션
- **핵심 시연**: 슬라이더로 수출량/환율 조정 → 차트 실시간 반응
- 비용 요약: 미제출 25.4억 → 실측 18.0억 → **절감 7.3억**
- "이 플랫폼으로 연간 7.3억 원을 절감할 수 있습니다"

### 4. 위성 환경 검증 (90초)
- 6개 년도 위성 썸네일 확인
- "CNN 분석 실행" 클릭 → 분석 진행
- 결과: 2019 vs 2024 위성 비교 (Segmentation + Grad-CAM)
- **핵심 시연**: 타임라인 재생 → 2019~2024 산림 변화 실시간
- NDVI 차트: 0.72 → 0.50 하락 추세
- Verdict: "HIGH RISK — 산림 31%p 감소"

### 5. 통합 보고서 (60초)
- "보고서 생성" 클릭
- 8개 섹션 스크롤:
  - Executive Summary: 7 PASS, 2 WARNING
  - 위성 사진 + CNN 결과 (대형)
  - CBAM 비용 표 + 절감액
  - 서류 검증 6건 전부 Valid
  - 9개 조항 상세 (배경/검증/판정/위험/권고)
- "DDS 보고서 내보내기" 클릭 → .txt 다운로드

## 핵심 어필 포인트
1. **"서류 올리면 끝"** — 5단계가 자동으로 이어짐
2. **"연 7.3억 절감"** — CBAM 슬라이더 시연이 가장 임팩트
3. **"위성 AI가 판별"** — 타임라인 재생이 시각적으로 강력
4. **"보고서 한방에"** — 기존 수 주 → 수 분
