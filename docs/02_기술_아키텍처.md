# 2. 기술 아키텍처

## 기술 스택

| 계층 | 기술 | 용도 |
|------|------|------|
| Frontend | React 18 + TypeScript | UI 프레임워크 |
| Build | Vite 6 | 번들러 |
| Styling | Tailwind CSS | 유틸리티 CSS |
| Animation | framer-motion | 페이지 전환, 마이크로 인터랙션 |
| Charts | Recharts | CBAM 시뮬레이션, NDVI 시계열 |
| Icons | lucide-react | 시스템 아이콘 |
| Crypto | Web Crypto API | SHA-256 블록체인 인증 |
| Satellite | Sentinel-2 Cloudless (EOX) | 년도별 위성 데이터 |
| CNN | Python (numpy, scipy, PIL) | Segmentation + Grad-CAM 생성 |

## 파이프라인 아키텍처

```
[Step 1] 서류 접수
    │ 드래그앤드롭 업로드, 6종 서류
    ▼
[Step 2] OCR 자동 추출
    │ AI 필드 추출 → 교차 검증 → HS코드 기반 규제 자동 매핑
    ▼
[Step 3] CBAM 탄소비용 산정
    │ 품질 스코어링 → 블록체인 인증 → 2026~2034 시뮬레이션
    ▼
[Step 4] 위성 환경 검증
    │ CNN Segmentation + Grad-CAM + NDVI 시계열
    ▼
[Step 5] 통합 컴플라이언스 보고서
    │ 전 단계 결과 종합 → 조항별 판정 → DDS 내보내기
```

## 위성 분석 파이프라인 상세

```
Sentinel-2 L2A (2019~2024)
    ↓
U-Net CNN (ResNet34 backbone)
    ↓ Semantic Segmentation
산림 / 농지 / 나지 / 도시 분류
    ↓
Grad-CAM Saliency Map
    ↓ 판단 근거 시각화
NDVI 시계열 (NIR−Red)/(NIR+Red)
    ↓
연도별 산림 비율 변화 산출
    ↓
EUDR Cutoff Date (2020-12-31) 대조
    ↓
최종 판정: High / Medium / Low Risk
```

## CBAM 비용 산정 모델

- **배출계수**: 3.2 tCO₂/t (Scope 1: 재배 1.42 + 정제 0.89 + 운송 0.54, Scope 2: 전력 0.35)
- **EU ETS 가격**: €87.5/tCO₂ (2026 Q1 기준, 연 8% 상승 가정)
- **무상할당 폐지**: 2026년 2.5% → 2034년 100%
- **시나리오**: 실측 제출 vs EU 기본값(4.5) vs 벤치마크(1.8)

## 데이터 흐름

1. 사용자가 서류 업로드
2. OCR이 HS코드 추출 → 품목별 적용 규제 자동 결정
3. CBAM: 배출계수 검증 + 블록체인 기록 + 비용 시뮬레이션
4. 위성: 원산지 GPS 좌표 기반 Sentinel-2 분석
5. 보고서: 모든 결과 종합 → PDF/TXT 내보내기
