1. 프로젝트 개요

프로젝트명

EcoGuard (구 EcoTrade)

팀

유니하나 (하나은행 대학생 연합)

한줄 요약

EU 환경 규제(EUDR·CBAM·CSDDD) 대응을 위한 AI 기반 통합 컴플라이언스 자동화 플랫폼

문제 인식

2023년부터 EU가 시행하는 3대 환경 규제로 인해, 한국 수출 기업은 다음 상황에 직면합니다:

  • EUDR (산림전용 규제): 2020년 이후 산림이 파괴된 토지의 원자재 EU 수입 금지. 위반 시 매출액 4% 과징금 + 시장 퇴출.
  • CBAM (탄소국경조정): 수입품에 내재된 탄소에 비용 부과. 2026년부터 실제 비용 발생, 2034년 100% 적용.
  • CSDDD (공급망 실사): 공급망 전체의 인권·환경 실사 의무. 매출액 5% 과징금.

현재 대부분의 기업은 이 규제들을 수동으로, 개별적으로 대응하고 있어 비용과 시간이 과다 소요됩니다.

솔루션

서류 제출 → OCR 자동 추출 → CBAM 탄소비용 산정 → 위성 기반 산림전용 검증 → 통합 DDS 보고서 생성까지 5단계 파이프라인을 하나의 플랫폼에서 자동화합니다.

핵심 차별점

  1. 위성 AI 검증: Sentinel-2 위성영상 + CNN(U-Net) + Grad-CAM으로 산림전용 여부를 자동 판별
  2. 비용 시뮬레이션: CBAM 무상할당 폐지 스케줄(2026~2034) 반영한 연도별 비용 예측
  3. 통합 보고서: 모든 규제를 한 보고서에서 조항별 판정 + 근거 + 권고사항 제공

시연 케이스

  • 품목: Crude Palm Oil (CPO), HS 1511.10
  • 경로: PT. Sawit Kalimantan Utama (인도네시아) → UniHana Trading GmbH (EU)
  • 수량: 2,400 MT
  • 원산지: Central Kalimantan, Indonesia (2.50°S, 111.79°E)

4. 핵심 기술 의사결정

GAN vs CNN 선택

초기 가설

GAN(Generative Adversarial Network)으로 위성 이미지를 생성/비교하여 산림전용을 검출하려 했습니다.

문제점

  • GAN은 이미지 "생성"에 특화 — 산림전용 "검출"과 목적이 다름
  • 재현성(reproducibility)이 낮아 규제 감사에 부적합
  • 생성된 이미지의 근거를 설명할 수 없음 (XAI 불가)

최종 결정: CNN 주력

  • U-Net (ResNet34): Semantic Segmentation으로 산림/농지/나지/도시 픽셀 단위 분류
  • Grad-CAM: CNN이 "어디를 보고 판단했는지" 히트맵으로 시각화 → 설명 가능한 AI
  • NDVI 시계열: 위성 밴드 연산으로 식생 건강도 정량화 → 수치 근거

선배(안지홍) 피드백

"CNN이 더 가볍고 편합니다. 애초에 CNN에 있는 CAM을 응용한 거라서 CNN쪽으로 가면 오히려 더 논문 주장은 강해요." "19년도 이미지랑 26년도 이미지 차이를 보는 게 좀 더 맞는 방법이라고 봅니다."

판정 엔진 비율

방법 비중 역할
CNN Segmentation + Grad-CAM 80% 주력 판정
GIS/지수 기반 Rule (NDVI 등) 15% 정량적 보조
GAN (향후) 5% 보조/미래 대비

위성 데이터 소스 선정

시도한 소스 결과 이유
Esri World Imagery ✗ 최신 1장만 제공, 년도별 불가
Esri Wayback ✗ 년도 무관 동일 이미지 반환
Google Earth Engine 미시도 API 키 필요, 복잡
Sentinel-2 Cloudless (EOX) ✓ 년도별 무료 WMS, 10m 해상도

파이프라인 순서 재구성

기존

서류 → OCR → 규제심사 → CBAM → 위성

문제

규제심사(Step 3)에서 "위성 결과 산림 31%p 감소"를 참조하는데, 위성 검증(Step 5)이 아직 안 끝남 → 논리적 모순

변경 후

서류 → OCR → CBAM → 위성 → 통합 보고서

이유: 모든 검증이 끝난 후에 통합 보고서로 결과를 종합하는 것이 논리적. 위성 검증 결과가 보고서의 EUDR 판정에 자동 반영됨.

UI/UX 디자인 원칙

  1. Monochrome: ink(#0A0A0A) + surface(#FAFAF9) + muted 계열만. 색은 차트 데이터에만.
  2. 사용자 주도: 자동 실행 ✗ → "분석 실행" 버튼 클릭 필요
  3. 경고박스 금지: 노란/파란 배경 카드 사용 안 함 (AI 생성 느낌)
  4. 절제: 큰 것보다 작고 정확하게. 과한 애니메이션/색상 자제.

5. 비즈니스 모델 및 시장 분석

타겟 시장

1차 타겟: 한국 → EU 수출 기업

  • 팜유, 목재, 커피, 카카오, 대두, 고무 (EUDR 7대 품목)
  • 철강, 알루미늄, 시멘트, 전력, 비료, 수소 (CBAM 6대 품목)
  • EU 매출 €150M 이상 기업 (CSDDD 대상)

시장 규모

  • EUDR 대상 한국 수출액: 약 $2.3B/년 (팜유 가공품, 목재 제품 등)
  • CBAM 대상 한국 수출액: 약 $8.7B/년 (철강, 알루미늄 등)
  • EU 수출 중소기업 약 3,200개사

가치 제안 (Value Proposition)

기업이 현재 겪는 문제

  1. 규제별로 다른 컨설팅 업체에 의뢰 → 비용 ×3
  2. 서류 준비 2~4주 소요 → 통관 지연
  3. 위성 검증 방법 부재 → EUDR 대응 불가
  4. CBAM 비용 예측 불가 → 가격 책정 실패

EcoGuard의 해결

  1. 3개 규제 통합 대응 → 원스톱
  2. AI OCR + 자동 검증 → 수 분 내 완료
  3. CNN 위성 분석 → 객관적 산림전용 판별
  4. 비용 시뮬레이션 → 사전 예측 가능

수익 모델

구분 가격 내용
Basic ₩500만/년 CBAM 보고서만
Pro ₩1,500만/년 EUDR + CBAM + 위성 분석
Enterprise 별도 협의 CSDDD 포함 + 커스텀 + API

경쟁 우위

기존 방식 EcoGuard

수동 서류 검토 (2~4주) AI OCR 자동 (수 분)

  • 실측 제출 시: 연 18.0억 원 (3.2 tCO₂/t)
  • 절감: 연 7.3억 원 (29%)
